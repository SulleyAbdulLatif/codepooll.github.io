    Creating a Basic Layout

    HTML layout can be achieved using various techniques, but one common approach is using CSS Grid. Below is a basic two-column layout using CSS Grid.

    <div class="container">
      <div class="sidebar">
        <!-- Sidebar content goes here -->
      </div>
      <div class="content">
        <!-- Main content goes here -->
      </div>
    </div>

    Responsive Layout

    CSS Grid allows for responsive layouts. In the example above, the layout changes to a single column on smaller screens using a media query.

    @media (min-width: 768px) {
      .container {
        grid-template-columns: 1fr 1fr;
      }
    }

    Custom Styling

    You can customize the sidebar and main content area styles as per your requirements. In this example, we set background colors, padding, and border for a clean look.

    .sidebar {
      background-color: #f2f2f2;
      padding: 20px;
    }

    .content {
      background-color: #fff;
      padding: 20px;
      border: 1px solid #ddd;
      border-radius: 4px;
    }

    Conclusion

    HTML layout is an essential aspect of web design. CSS Grid provides a powerful and flexible way to create responsive and visually appealing layouts for your web pages.
