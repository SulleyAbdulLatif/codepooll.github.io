    The <ul> and <li> Tags

    The <ul> and <li> tags are used to create unordered lists. The <ul> tag represents the list container, and the <li> tags represent the list items.

    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
    </ul>

    Nested Lists

    You can create nested lists by placing a new <ul> or <ol> (ordered list) inside an <li> element of an existing list.

    <ul>
      <li>Item 1</li>
      <li>Item 2
        <ul>
          <li>Subitem 2.1</li>
          <li>Subitem 2.2</li>
        </ul>
      </li>
      <li>Item 3</li>
    </ul>

    The <ol> Tag

    The <ol> tag is used to create ordered lists. It represents a numbered list, where the order of items is important.

    <ol>
      <li>First item</li>
      <li>Second item</li>
      <li>Third item</li>
    </ol>

    List Attributes

    Lists in HTML can have several attributes, such as `type` and `start` for ordered lists, and `value` for list items in ordered lists.

    <ol type="A" start="3">
      <li value="3">Third item</li>
    </ol>

    Conclusion

    HTML lists are useful for structuring content in an organized and easy-to-read manner. Unordered and ordered lists allow you to present information with bullet points or numbering based on your content's requirements.
