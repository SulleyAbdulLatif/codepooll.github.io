    Creating a Basic Table

    Tables in HTML are created using the <table> element. Use <tr> for table rows, <th> for table headers (in the first row), and <td> for table data cells.

    <table>
      <tr>
        <th>Name</th>
        <th>Age</th>
      </tr>
      <tr>
        <td>John</td>
        <td>25</td>
      </tr>
      <tr>
        <td>Jane</td>
        <td>30</td>
      </tr>
    </table>

    Table Attributes

    Tables can have attributes like `border`, `cellspacing`, and `cellpadding`. However, these attributes are not recommended, and you should use CSS for styling instead.

    <table border="1" cellspacing="0" cellpadding="10">
      <!-- table content -->
    </table>

    Spanning Columns and Rows

    You can merge cells to span multiple columns or rows using the `colspan` and `rowspan` attributes.

    <table>
      <tr>
        <th colspan="2">Personal Information</th>
      </tr>
      <tr>
        <td>Name</td>
        <td>Age</td>
      </tr>
      <tr>
        <td>John</td>
        <td>25</td>
      </tr>
    </table>

    Styling Tables with CSS

    Use CSS to style tables, including adding borders, changing background colors, and adjusting cell padding.

    <style>
    table {
      border-collapse: collapse;
      width: 100%;
      margin-bottom: 20px;
    }
    th, td {
      border: 1px solid #ddd;
      padding: 8px;
      text-align: left;
    }
    th {
      background-color: #f2f2f2;
    }
    </style>

    Conclusion

    HTML tables are useful for displaying data in a structured manner. Properly organizing data using tables improves readability and user experience on web pages.
