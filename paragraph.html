    The <p> Tag

    The <p> tag is used to define paragraphs of text on a web page. It represents a block of text with its own line breaks.

    <p>This is a paragraph of text. Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>

    Paragraph Formatting

    Paragraphs are automatically formatted with some default styles, such as having a margin and line spacing. You can further style paragraphs using CSS to change the font size, color, alignment, and more.

    Line Breaks

    By default, the <p> tag adds space between paragraphs. If you want to add line breaks within a paragraph without starting a new paragraph, you can use the `
    ` tag.

    <p>This is the first line.<br>This is the second line within the same paragraph.</p>

    Whitespace and Indentation

    HTML ignores multiple spaces and line breaks within a paragraph. If you want to preserve extra spaces or add indentation, you can use the <pre> tag or apply CSS styles.

    <pre>This is a paragraph with indentation.</pre>

    Conclusion

    Understanding how to create and format paragraphs is essential for organizing and presenting content on your web pages. The <p> tag is the foundation for creating structured and readable text blocks.
