    The <h1> Tag

    The <h1> tag is used to define the top-level heading of a web page. It represents the most important heading and should be used for the main title of the page.

    <h1>This is the Main Heading</h1>

    The <p> Tag

    The <p> tag is used to define paragraphs of text on a web page. It represents a block of text with its own line breaks.

    <p>This is a paragraph of text. Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>

    The <a> Tag

    The <a> tag is used to create hyperlinks or anchor links. It allows you to link to other web pages or resources.

    <a href="https://example.com">Click here to visit Example.com</a>

    The <img> Tag

    The <img> tag is used to embed images on a web page. It requires a src attribute that specifies the image file's URL.

    <img src="image.jpg" alt="Image description">

    The <ul> and <li> Tags

    The <ul> and <li> tags are used to create unordered lists. The <ul> tag represents the list container, and the <li> tags represent the list items.

    <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
    </ul>

    The <table> Tag

    The <table> tag is used to create tables on a web page. It contains one or more <tr> (table row) elements, and each row can contain <td> (table data) or <th> (table header) elements.

    <table>
      <tr>
    <th>Name</th>
    <th>Age</th>
    </tr>
    <tr>
    <td>John</td>
    <td>25</td>
    </tr>
    <tr>
    <td>Jane</td>
    <td>30</td>
     </tr>
    </table>

    The <div> Tag

    The <div> tag is a generic container element that is used to group other elements together. It is often used for styling purposes or to create layout structures.

    <div class="container">
    <p>This is inside a div container.</p>
    </div>

    Conclusion

    These are some of the essential HTML elements and tags that you'll commonly use when creating web pages. Understanding how each tag works and its purpose is crucial for building well-structured and organized web content.
