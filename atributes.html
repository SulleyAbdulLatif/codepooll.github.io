    The `src` Attribute

    The `src` attribute is used with the <img> tag to specify the URL of the image to be displayed on the web page.

    <img src="image.jpg" alt="Image description">

    The `href` Attribute

    The `href` attribute is used with the <a> tag to specify the URL of the target web page or resource.

    <a href="https://example.com">Click here to visit Example.com</a>

    The `alt` Attribute

    The `alt` attribute is used with the <img> tag to provide alternative text for the image. It is displayed if the image cannot be loaded or for accessibility purposes.

    <img src="image.jpg" alt="Image description">

    The `style` Attribute

    The `style` attribute is used with various HTML tags to apply inline CSS styles to specific elements.

    <p style="color: #ff0000;">This paragraph has red text color.</p>

    The `class` Attribute

    The `class` attribute is used to assign one or more class names to an HTML element. It allows you to apply styles or scripts to elements with the same class.

    <p class="highlight">This paragraph has a custom style applied through the "highlight" class.</p>

    The `id` Attribute

    The `id` attribute is used to uniquely identify an HTML element within a web page. It is often used for CSS styling or JavaScript interactions.

    <div id="header">This is the header content</div>

    The `disabled` Attribute

    The `disabled` attribute is used with form elements such as buttons and input fields to disable user interaction.

    <button disabled>This button is disabled</button>

    Conclusion

    These are some of the essential HTML attributes that you'll commonly use when building web pages. Attributes provide additional information and behavior to HTML elements, enhancing the functionality and appearance of your web content.
