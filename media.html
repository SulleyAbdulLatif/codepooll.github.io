    Embedding Images

    To display images on a web page, use the <img> tag with the `src` attribute specifying the image URL.

    <img src="image.jpg" alt="Image description">

    Embedding Videos

    To embed videos, use the <video> tag with the `src` attribute specifying the video URL and the `controls` attribute to add playback controls.

    <video src="video.mp4" controls></video>

    Embedding Audio

    To embed audio, use the <audio> tag with the `src` attribute specifying the audio URL and the `controls` attribute to add audio controls.

    <audio src="audio.mp3" controls></audio>

    Responsive Media

    To make media elements responsive, use CSS with the `max-width` property.

    <style>
    img, video, audio {
      max-width: 100%;
      height: auto;
    }
    </style>

    Alternative Content

    Always provide alternative content for media elements in case they cannot be displayed. Use the `alt` attribute for images and content between the opening and closing tags for videos and audio.

    <img src="image.jpg" alt="Image description">
    <video src="video.mp4" controls>Your browser does not support the video tag.</video>
    <audio src="audio.mp3" controls>Your browser does not support the audio tag.</audio>

    Conclusion

    HTML provides simple and powerful elements for embedding media on web pages. Use the appropriate tags and attributes to enhance the user experience and accessibility of your multimedia content.
